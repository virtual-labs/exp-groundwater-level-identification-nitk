Geo-resistivity test for Groundwater level identification

Objective:

Determination of ground water level by using georesistivity method.


Apparatus used:

Resistivity meter, cables, battery, hammer, measuring tape etc.


 Description

Georesistivity Test: The electrical resistivity test is useful for studying the existence of subsurface structures by observing the differences in their resistance to electrical current flow and hence determining the presence of groundwater.

Variables Used in Calculations

AB= Distance between the current electrodes

MN = Distance between the potential electrode

Resistivity= Value from the resistivity meter

K= Hydraulic conductivity

K = 3.14*( (AB2 )2- (MN⁄2)2 (2*MN2 ) )

Apparent Resistivity= K * Resistivity
